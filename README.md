## 📋 What is demo-repo-applicaiton?

demo-repo-applicaiton is a simple desktop app made to help users explore complex data through clear and interactive visuals. This app shows military equipment transfers, heart signals from ECGs, healthcare text analysis, and maps of military bases all in one place.

You do not need any technical knowledge or coding skills. The app uses easy charts, maps, and dashboards you can click and explore to learn from the data.

## 🖥️ System Requirements

To run demo-repo-applicaiton smoothly, make sure your computer meets these requirements:

- **Operating System:** Windows 10 or later, macOS 10.14 or later, or Linux (Ubuntu 18.04+)
- **Processor:** Intel Core i3 or equivalent
- **Memory:** At least 4 GB of RAM
- **Storage:** Minimum 500 MB free space
- **Internet Connection:** Required for downloading the app and some live data features

This app uses the Python environment behind the scenes but packages everything so that you don’t have to install or know about Python.

## 🚀 Quick Start Guide

Once you open the app, you will see four main sections:

- **Military Equipment Transfers**: Interactive charts showing where and when military items were transferred across countries.
- **ECG Signal Analysis**: Visual displays of heart signals to help understand heart health patterns.
- **Healthcare Document NLP**: Breaks down healthcare documents into easy insights using natural language processing.
- **Military Base Mapping**: Maps that show locations and details of military bases around the world.

Use your mouse or touchpad to click on buttons, charts, or maps. Try zooming or hovering over parts of the dashboard to see details.

No coding is required. Everything works with simple clicks and menus.

## 🛠 Features

- **Interactive Dashboards:** Change views by selecting options or clicking on data points.
- **Clear Visuals:** Uses charts and maps designed for easy understanding.
- **Multi-Topic Data:** Combines military, healthcare, and health monitoring data in one place.
- **Offline Use:** Once installed, you can work without internet access for most functions.
- **Export Options:** Save charts and data as images or CSV reports.
- **Responsive Design:** Works on different screen sizes and laptops.

## 📱 How to Use Each Section

### Military Equipment Transfers

- Explore data by choosing countries or time periods.
- View flow lines on maps to see equipment moving paths.
- Use filters to focus on certain types of military gear.

### ECG Signal Analysis

- Load ECG data from sample files included.
- Watch the heartbeats animated on the screen.
- Get simple stats like heart rate or irregularities.

### Healthcare Document NLP

- Upload or browse healthcare documents.
- See summaries and keyword highlights.
- Check sentiment and category tags assigned automatically.

### Military Base Mapping

- Browse bases by region or size.
- Click markers on the map to get base details.
- Switch between map views like satellite or terrain.
